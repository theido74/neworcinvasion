This game inspired by Space Invaders set in the Lord of the Rings universe. The first chapter consists of five levels, you can launch the game from the "testsqlite.py" file, register to keep your high score, and use the [S] key to start, [SPACE] key to shoot, and arrow keys for movement.

Required libraries:

PIL
Random
SQLite3
bcrypt
pygame
Tkinter

A patch file has been added to ensure paths with [] valid under Windows can be read under Linux or Mac.

Looking forward to your feedback, and have a great gaming experience!
